# 🎓 SmartCollege Monitor

Веб-приложение для мониторинга успеваемости студентов, созданное на Python с использованием Streamlit.

---

## 🚀 О проекте

SmartCollege Monitor — это интерактивная система для:
- добавления оценок студентов
- хранения данных (JSON)
- анализа успеваемости
- визуализации статистики (графики и рейтинги)
- фильтрации по группам и предметам

---

## 🛠️ Технологии

- Python 🐍
- Streamlit
- Pandas
- Plotly
- JSON (для хранения данных)

---

## 📊 Основной функционал

- ➕ Добавление оценок студентов
- 📋 Таблица всех оценок
- 📈 Графики успеваемости
- 🏆 Рейтинг студентов
- 🔍 Фильтрация по группам и предметам
- 💾 Сохранение данных между запусками (students.json)
- ❌ Удаление записей
